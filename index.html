What is code?

For those that don't know, code is a language in which computer programs, apps and websites are written. Just as there are many languages spoken around the world, there are also a variety of different programming languages.

Why learn to code?

According to the government's National Career Service, starting salaries for computer programmers in the games industry start at around £25,000, rising to between £30,000 and £50,000 with more experience and responsibility. As more and more jobs are created in the tech industry, learning to code will also open up more employment opportunities.

How long does it take?

The length of time it takes to learn to code all depends on the person and how much time they're willing/able to out in. Learning how to build simple websites, for example, can be learnt in a matter of weeks.
